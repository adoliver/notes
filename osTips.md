## Filesystem

## Terminal

[TMux](OsTips/tmux.md) is a windowed terminal multiplexer, allowing multiple terminals in a single window.
* Installation
	* Use ```brew install tmux``` and add ```tmux new -A -s main``` to run on startup in the terminal profile
